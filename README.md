# Velvet — Deploy from Your Phone

## The easiest way: StackBlitz (no laptop needed)

StackBlitz lets you run and deploy this app entirely from your phone browser.

---

## Step 1 — Create a GitHub account (free)
Go to https://github.com and sign up.
GitHub is where your code will be stored — it's free and takes 2 minutes.

---

## Step 2 — Create a new repository
1. Once logged in, tap the **+** icon (top right)
2. Tap **New repository**
3. Name it: `velvet`
4. Set it to **Public**
5. Tap **Create repository**

---

## Step 3 — Upload your files
1. On your new repository page, tap **uploading an existing file**
2. Upload ALL the files from this zip, keeping the folder structure:
   - `index.html`
   - `package.json`
   - `vite.config.js`
   - `src/main.jsx`
   - `src/App.jsx`
3. Tap **Commit changes**

---

## Step 4 — Deploy on Vercel (from your phone)
1. Go to https://vercel.com and sign up using your GitHub account
2. Tap **Add New Project**
3. You'll see your `velvet` repository — tap **Import**
4. Leave all settings as default
5. Tap **Deploy**
6. In about 60 seconds you'll get a live link like: **https://velvet.vercel.app**

---

## That's it!
You now have a live app you can share with anyone from your phone.
Send that link in your pitch emails as your demo.

---

## To make changes later
Edit `src/App.jsx` directly on GitHub from your phone browser,
then go to Vercel and it will automatically redeploy.
